# Akıllı Kütüphane Otomasyon Sistemi
Bu proje, Kastamonu Üniversitesi Tosya MYO Programlama II dersi için hazırlanmıştır.

## Özellikler
- **OOP Yapısı:** Modüler ve sınıflar arası ilişki tabanlı tasarım.
- **ISBN Kontrolü:** Mükerrer kayıt engelleme sistemi.
- **Fiziksel Takip:** Raf ve sıra numarası yönetimi.
- **Otomatik Kayıt:** Her işlem sonrası verileri `.txt` dosyasına kalıcı kaydeder.

## Kullanım
1. Python yüklü olduğundan emin olun.
2. `main.py` dosyasını çalıştırın.
